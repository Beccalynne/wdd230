Typography

[each font and where it will be used, even add font sizes and color for each element]

Headings

Source Serif Pro

This will be used with any heading that is used, on the top of the page or each section heading. I am going to use the same font size that is places it in with it being a heading. I will use the color hex #1F363D expect for in the heading where the title of the page will be #D9D9D9 but with the same font.

Paragraph

Oxygen

This font will be used for majority of the writing on the page. If anything needs to be highlighted then I will use a bold or italics to indicate that. I will use a dark color that has a hex color of #1F363D.

Captions

Bebas Neue

This I will use beneath any photo, graph, or anything that will need a caption. I am going to use an accent color for this font that is called rust with a hex of #BA3F1D.
